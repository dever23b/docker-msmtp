<h1 align="center">docker-msmtp</h1>

### About

SMTP relay server using [msmtpd](https://marlam.de/msmtp). Written on top of [LinuxServer.io](https://www.linuxserver.io) base Alpine [image](https://github.com/linuxserver/docker-baseimage-alpine) for an experience consistent with other LinuxServer images, including customizable non-root user support.

> Inspired by [CrazyMax](https://github.com/crazy-max)'s [docker-msmtpd](https://github.com/crazy-max/docker-msmtpd) project.

> More info: [MSMTP Documentation](https://marlam.de/msmtp/documentation/)
